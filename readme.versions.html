                    ** Set up
                    
                    DROP_BOX Config
                    1. In "projects/config/" create dropbox_dev.ini similar to "dropbox.ini".
                    2. In settings_dev, define 
                    DROP_BOX_CONFIG_FILENAME = 'dropbox_dev.ini'
                    
                    Dropbox - Create a symlink to Dropbox/directory photos in media folder that will contain all photographers directories
                    
                    Celery
                    
                    Run Celery in dev environment
                    
                    python manage.py celeryd -l info
                    
                    Run CeleryBeat in dev environment for periodic tasks
                    python manage.py celerybeat

                    ** Image Modifications
                    
                    Generating Preview Images
                    
                    create_preview_image - Task create_preview_image generates the color and b/w preview images. 
                    It calls shell script create_preview_image.sh to generate color and b/w preview images and 
                    then creates new instances of PhotoFile to store them in db.
                    
                    Generating B/W Version
                    
                    create_bw_image - Task generates a b/w version of the image by calling shells cript create_bw_image.sh. 
                    Then creates new instance of PhotoFile and saves in DB.
